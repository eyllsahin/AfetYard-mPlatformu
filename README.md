# 🆘 AfetYardım Platformu | Disaster Relief Platform

> **[TR]** Afet anlarında ihtiyaç sahiplerini gönüllü destekçilerle buluşturan kriz koordinasyon platformu.  
> **[EN]** A crisis coordination platform that connects those in need with volunteer supporters during disasters.

---

## 🇹🇷 Türkçe

### 📖 Hakkında

**AfetYardım Platformu**, deprem ve doğal afet sonrasında ortaya çıkan ihtiyaçları hızlı bir şekilde organize etmek ve gönüllü destekçilerle buluşturmak için geliştirilmiş bir web uygulamasıdır. Platform, bilgi kirliliğini ortadan kaldırarak gerçek ihtiyaçların görünür olmasını sağlar.

### ✨ Özellikler

- **📋 İlan Sistemi** — İhtiyaç ve destek ilanları oluşturma ve görüntüleme
- **🔴 Acil Durum Önceliklendirme** — Acil ilanlar otomatik olarak öne çıkarılır
- **🔍 Gelişmiş Arama** — Konum, başlık ve detaya göre arama yapabilme
- **🏷️ Kategori Filtreleri** — Gıda, Barınma, Sağlık, Eğitim, Lojistik, Psikoloji, Hayvanlar, Eşya, Çocuk ve İş kategorileri
- **✅ Görev Üstlenme** — İlanları üstlenip takip edebilme
- **📱 Responsive Tasarım** — Mobil ve masaüstü uyumlu
- **🌙 Karanlık / Aydınlık Tema** — Kullanıcı tercihine göre tema değiştirme
- **📴 Çevrimdışı Destek** — LocalStorage ile çevrimdışı veri erişimi
- **🎨 Modern Arayüz** — Aurora arkaplan efektleri ve cam (glass) tasarım sistemi

### 🚀 Kurulum

1. Projeyi klonlayın:

## 🇬🇧 English

### 📖 About

**AfetYardım Platform** (Disaster Relief Platform) is a web application developed to quickly organize the needs that arise after earthquakes and natural disasters and to match them with volunteer supporters. The platform eliminates information pollution and ensures real needs are visible.

### ✨ Features

- **📋 Listing System** — Create and view need and support listings
- **🔴 Emergency Prioritization** — Urgent listings are automatically highlighted
- **🔍 Advanced Search** — Search by location, title, and details
- **🏷️ Category Filters** — Food, Shelter, Health, Education, Logistics, Psychology, Animals, Supplies, Children, and Jobs categories
- **✅ Task Assignment** — Take on and track listings
- **📱 Responsive Design** — Mobile and desktop compatible
- **🌙 Dark / Light Theme** — Theme switching based on user preference
- **📴 Offline Support** — Offline data access via LocalStorage
- **🎨 Modern UI** — Aurora background effects and glassmorphism design system

### 🗂️ Categories

| Emoji | Category | Description |
|-------|----------|-------------|
| 🍞 | Food | Food and beverage needs |
| 🏠 | Shelter | Temporary or permanent housing |
| 🏥 | Health | Medicine, blood donation, medical supplies |
| 🐾 | Animals | Support for stray animals |
| 📚 | Education | Educational support and materials |
| 📦 | Supplies | Clothing, furniture, electronics |
| 🧠 | Psychology | Psychological support and counseling |
| 🚚 | Logistics | Transportation, storage, energy |
| 🧒 | Children | Activities and support for children |
| 💼 | Jobs | Job opportunities |

### 🚀 Installation

1. Clone the project:
   ```bash
   git clone https://github.com/eyllsahin/AfetYard-mPlatformu.git
   ```
2. Navigate to the project directory:
   ```bash
   cd AfetYard-mPlatformu
   ```
3. Open `index.html` in a browser or start a local server (e.g., Live Server extension).

### 🛠️ Technologies Used

- **HTML5** — Structure
- **CSS3** — Styling (Aurora effects, Glass Morphism, CSS variables)
- **Vanilla JavaScript** — Application logic, filtering, LocalStorage management
- **Google Fonts** — Poppins & JetBrains Mono
